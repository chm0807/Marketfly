# Marketfly
Es una app de caja/POS para supermercado. Permite escanear productos con lector HID o serial, armar la factura, calcular totales, elegir pago (tarjeta/QR/Nequi) y guardar compras favoritas por cédula. Está hecha con React+TypeScript+Vite, empaquetada con Tauri y usa Rust para el lector serial.
